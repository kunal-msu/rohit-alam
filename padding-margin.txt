1.The CSS padding properties are used to generate space
 around an element's content, inside of any defined borders.

2.CSS has properties for specifying the padding for each side of an element:

.padding-top
.padding-right
.padding-bottom
.padding-left.

2Q Margins
1.The CSS margin properties are used to create space around elements,
 outside of any defined borders.
With CSS, you have full control over the margins. 
There are properties for setting the margin for each side of an element (top, right, bottom, and left).
2.CSS has properties for specifying the margin for each side of an element:

.margin-top
.margin-right
.margin-bottom
.margin-left.
